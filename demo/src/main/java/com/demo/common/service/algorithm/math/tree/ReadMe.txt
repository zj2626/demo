树：

结点的度：  该节点有几个子节点（0-叶子节点）
树的度：    树中节点最大有几个子节点
结点的层次： 从根结点开始定义的层次
树的深度：  最大层次叫深度

满二叉树    所有的非叶子结点都有两个子结点
完全二叉树： 最下层从最右边去掉相邻的部分叶子结点的二叉树
满二叉树是完全二叉树；完全二叉树不一定是满二叉树

性质：
1. 二叉树第i层最多有2^(i-1)个结点
2. 高度为h的二叉树最多有(2^h)-1个结点
3. 任意二叉树：叶子结点数 = 度数为2的结点数目 + 1

遍历:
1. DLR: 先序遍历: 根 左子树 右子树     :1.递归 2.非递归
2. LDR: 中序遍历: 左子树 根 右子树     :1.递归 2.非递归
3. LRD: 后序遍历: 左子树 右子树 根     :1.递归 2.非递归
4. 按层遍历                          :非递归
